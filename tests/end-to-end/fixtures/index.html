Ariadne Browser Test Suite

This page lists all available test fixtures for the Ariadne semantic HTML extraction library.

  • Simple Form Test
    Basic form with various input types
  • Complex Layout Test
    Tests extraction from complex nested structures
  • Large Document Test
    Tests token budget limits and performance
  • Dynamic Content Test
    Tests detection of lazy loading and infinite scroll
  • IFrame Test
    Tests cross-origin iframe handling
  • Accessibility Test
    Tests extraction of ARIA labels and accessibility features
  • Edge Cases Test
    Tests handling of malformed HTML and edge cases
